**Install**:
```bash
npm i -g hexlet-brain-games-nor
```
**Usage**:

- ```brain-calc```

- ```brain-even```

- ```brain-prime```

- ```brain-gcd```

- ```brain-progression```

**Description**:
  
In the project it was necessary to implement a set of mini-games launched from the console. These games are similar to what is called "Brain Games".
